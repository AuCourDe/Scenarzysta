# System Generujący Scenariusze Testowe

## Informacje o Projekcie

Ten projekt to system generujący scenariusze testowe na podstawie multimodalnej dokumentacji.

## Status

Aktualnie projekt jest w fazie mockup. Plik `main.py` zawiera jedynie szkielet aplikacji.

## Dokumentacja

Wszelkie informacje potrzebne do działania aplikacji znajdują się w pliku:
- `Projekt Realizacyjny_ System Generujący Scenariusze Testowe na Podstawie Multimodalnej Dokumentacji.md`

Ten dokument zawiera szczegółowy opis:
- Architektury rozwiązania (lokalnego i chmurowego)
- Procesu ekstrakcji i analizy multimodalnych danych
- Weryfikacji dokumentacji i generowania scenariuszy testowych
- Implementacji eksportu do formatu Excel
- Zarządzania stanem aplikacji

## Wymagania

- Python 3.x
- Wirtualne środowisko Python (venv)

## Instalacja

1. Utwórz i aktywuj wirtualne środowisko:
```bash
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# lub
venv\Scripts\activate  # Windows
```

2. Zainstaluj wymagane zależności (po ich zdefiniowaniu w requirements.txt)

## Uruchomienie

```bash
python main.py
```
